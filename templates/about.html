About Intrusion Detection System

Intrusion Detection System (IDS) is a cybersecurity tool that monitors network traffic and detects suspicious activities or attacks.

This project uses Machine Learning algorithms trained on network traffic data such as:

  • Protocol Type
  • Service
  • Source & Destination Bytes
  • Flags and Network Behavior

The system classifies traffic into:

Normal Traffic Intrusion Attack
