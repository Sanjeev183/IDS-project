# Intrusion Detection System
Welcome to my "Machine Learning-based Intrusion Detection System"
This project is designed to detect whether network activity is "Normal" or an "Attack", helping improve cybersecurity using ML.

#Project overview
This IDS uses a *Random Forest Classifier* to analyze network traffic data and classify it into:
  Normal Activity
  Intrusion / Attack

It also provides a "confidence level" for each prediction through a web interface.

# Features
- Machine Learning-based detection  
- Real-time prediction via web interface  
- Confidence level display  
- Clean and simple UI  
- Trained on labeled dataset
